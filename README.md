## 📝 Descripción

El proyecto **PC-GAMER** tiene como objetivo diseñar una solución que ayude a los usuarios a conocer configuraciones de computadores gamer según sus necesidades y presupuesto. En esta primera entrega se presenta el avance del proyecto, incluyendo el informe y la planificación del desarrollo, siguiendo los lineamientos de la asignatura **Arquitectura de Hardware**.

## 📂 Contenido del repositorio
- 📄 Informe del proyecto en formato PDF.
- 📦 Archivo ZIP con el avance del proyecto.
- 📋 Archivo README con la información general del proyecto.

## 🎯 Objetivo
Desarrollar un proyecto que permita orientar a los usuarios en la selección de componentes de hardware para un computador gamer, buscando un buen equilibrio entre rendimiento, compatibilidad y costo.
